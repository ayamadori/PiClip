PiClip
========

A simple Japanese input helper.
http://store.ovi.com/content/569913

1. Feature
-----------
* Flick keyboard
* Word & Phrase prediction
* Emoji (EXPERIMENTAL)
* Open browser
* [Share] menu *UPGRADE only
* Clipboard (EXPERIMENTAL)

2. Using Nokia APIs
-------------
* Share API
* Nokia Gesture API
* Nokia UI API
* Nokia In-App payment plug-in

3. Test
-----
Tested on Nokia Asha SDK 1.1 & 1.2(beta).
