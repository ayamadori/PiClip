節約辞書 Ver.0.1
=================
2006年9月20日

##これは何？
　節約辞書は Windows CE 系オペレーティングシステム（それはしばしばPocketPC、Windws Mobile などの製品名で呼ばれる）上で動作する POBox 実装のための変換辞書です。[富豪辞書 for CE](http://fepunch.hp.infoseek.co.jp/omake/fugodic4ce.html)では大きすぎる、PDA ではそれほど入力しないのでできるだけ容量を節約したいという人のためにより小さくすることを意図しています。

　節約辞書は[Palm 版 POBox 向け辞書 PB-tk r5](http://www001.upp.so-net.ne.jp/focus/)をもとに、富豪辞書 for CE 開発成果の主要部分を流用し、900KB 程度のデータ量に収めることを目指しています（例文辞書は富豪辞書 for CE と同じものです）。

　このリリースは開発版です。今後大きく内容が変わる可能性があります。

##配布内容
+words.txt	…単語辞書
+phrases.txt	…例文辞書
+gpl.txt	…GNU GPL 原文
+gpl(j).txt	…GNU GPL 和訳文
+readme.txt	…このファイル。
